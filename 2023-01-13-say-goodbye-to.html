2023-01-13 Say Goodbye to ../../../

几年前还用 Babel 时，我写过一篇文章 2018-08-16 Say Goodbye to '../' '../../' '../../../'。JS 社区的工具链真的发展的好快，我已经很久没用过 Babel 了，😂

我现在主要是用 esbuild + typescript，因为 TypeScript 也变强了，所以现在配置绝对路径也变的简单了。

  • 不需要再依赖 loader 了
  • 只需要配置 tsconfig.json 即可
{
  "compilerOptions": {    
    "baseUrl": ".",
    "paths": {
      "~/*": ["src/*"]
    }
  }
}

so easy.
