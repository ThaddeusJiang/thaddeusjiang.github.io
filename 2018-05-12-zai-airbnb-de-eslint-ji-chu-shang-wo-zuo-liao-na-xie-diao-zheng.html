2018-05-12 在 airbnb 的 eslint 基础上，我做了哪些调整

没有完美的解决方案，适合自己的才是最好的。

TJ

无分号

"semi": ["warn", "never"],

允许 console 输出 warn error

"no-console": ["error", { "allow": ["warn", "error"] }],

允许简单运算符混合运用（ + - \* / 等），依旧禁止复杂运算符混合运用

"no-mixed-operators": [
    "error",
    {
        "groups": [
            ["&", "|", "^", "~", "<<", ">>", ">>>"],
            ["==", "!=", "===", "!==", ">", ">=", "<", "<="],
            ["&&", "||"],
            ["in", "instanceof"]
        ],
        "allowSamePrecedence": true
    }
  ],
