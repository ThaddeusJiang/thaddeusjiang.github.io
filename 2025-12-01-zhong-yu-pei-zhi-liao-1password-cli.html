2025-12-01 终于配置了 1password cli

随着 CLI AI tools 用的越来越多，越来越担心自己的环境变量被盗用了。虽然还没有到亡羊补牢，但是也不算早了。

建议：所有人都不要将 API keys 和密钥通过 plaintext 保存。

极简记录配置 GitHub CLI gh 使用 1password 的过程

1. 安装 1password cli

brew install 1password-cli
op --version

2. 安装 1password cli 插件

op plugin init gh
echo "source ~/.config/op/plugins.sh" >> ~/.zshrc && source ~/.zshrc

3. 删除 1password 以外副本

rm ~/.config/gh/hosts.yml
